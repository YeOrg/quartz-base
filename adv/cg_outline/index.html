    # Houdini Engine

    Install

    在编辑器Project->Assets中，Import Package ->Custom Package…，选择到Houdini安装目录(engine/unity/HoudiniEngineUnity.unitypackage) 或者也可以从Github仓库 GitHub - sideeffects/HoudiniEngineForUnity: Houdini Engine for Unity安装。

    https://www.sidefx.com/products/houdini-engine/plug-ins/

    # USD

    Installing

    GitHub - Unity-Technologies/usd-unity-sdk: Integration of Pixar’s Universal Scene Description into Unity

    • installing by name, “com.unity.formats.usd”;
    • installing by Git URL, “ https://github.com/Unity-Technologies/usd-unity-sdk.git?path=package/com.unity.formats.usd”;
    • or browsing for a local package with path <root>/package/com.unity.formats.usd/package.json.
